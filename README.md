# Android Obfuscation 

This repository provides an overview on tools, techniques and papers related to Android obfuscation.


---

## Background 
The repository was created as part of my Bachelors thesis on Android malware obfuscation & deobfuscation techniques, written at the Brandenburg University of applied Sciences in cooperation with the [RSF Digital Security Lab](https://rsf.org/en/digital-security-lab).
